# Assets needed before launch

These files are referenced in `index.html` and tool pages but are **not yet in the repo**. Add them to the project root before going live.

| File | Size | Purpose |
|------|------|---------|
| `favicon.ico` | 32×32 or multi-size | Browser tab icon |
| `favicon-16x16.png` | 16×16 | Small favicon |
| `favicon-32x32.png` | 32×32 | Standard favicon |
| `apple-touch-icon.png` | 180×180 | iOS home screen |
| `og-image.png` | **1200×630** | Open Graph / Twitter card preview |

## OG image spec

- Dimensions: 1200×630 px
- Background: dark (#03030a) with GeetAI gradient accent
- Text: "GeetAI Studio — 30 AI-Powered Tools for Business Growth"
- Logo: place GeetAI wordmark if available

## Quick placeholder

Until a designed asset exists, generate a 1200×630 PNG with the GeetAI brand colors and save as `og-image.png` in the repo root. All `<meta property="og:image">` tags already point to `https://geetaistudio.com/og-image.png`.
